# EXECUTIVE DEMO TALK TRACK

## Voiceover

### 0s-8s

`고객은 원하는 상품을 검색해서 바로 탐색을 시작할 수 있습니다.`

### 8s-18s

`검색 결과에서 상품을 확인하고, 상세 페이지에서 장바구니에 담습니다.`

### 18s-33s

`장바구니에서 바로 주문으로 이동하고, 체크아웃을 마치면 주문이 생성됩니다.`

### 33s-43s

`이 주문은 고객 경험에서 끝나지 않고 곧바로 운영 화면으로 이어집니다.`

### 43s-55s

`관리자는 관리자 콘솔에서 방금 들어온 주문을 즉시 확인할 수 있습니다.`

### 55s-60s

`그리고 같은 콘솔에서 상품 운영까지 이어서 처리할 수 있습니다.`

## Caption Script

1. `고객은 상품을 검색하고`
2. `상세 페이지에서 장바구니에 담습니다`
3. `장바구니에서 주문으로 이어지고`
4. `주문이 완료되면`
5. `관리자는 관리자 화면에서 즉시 확인합니다`
6. `같은 콘솔에서 상품 운영까지 이어집니다`

## Short Closing Line

`즉, 고객의 주문이 관리자 운영으로 바로 연결됩니다.`

## Delivery Notes

- 기능 설명보다 결과 설명으로 말한다.
- “제공합니다”보다 “방금 생성되었습니다 / 확인됩니다” 같은 현재형을 쓴다.
- 자막은 1문장 14자~24자 안쪽으로 짧게 유지한다.
