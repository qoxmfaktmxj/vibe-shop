# EXECUTIVE DEMO CHECKLIST

## Before Recording

- [ ] `http://127.0.0.1:8180/actuator/health` 가 `UP` 이다
- [ ] `http://127.0.0.1:4100` 이 열린다
- [ ] 관리자 계정 `admin@maru.local / admin1234!` 로그인 가능
- [ ] 상품 `linen-bed-set` 또는 `brew-mug` 가 실제로 보인다
- [ ] 장바구니가 비어 있다
- [ ] 브라우저 확대 비율이 `100%` 다
- [ ] 불필요한 탭과 알림을 모두 닫았다
- [ ] 네트워크가 안정적이다

## Right Before Recording

- [ ] 고객 탭: 홈 화면 열기
- [ ] 관리자 탭: 로그인 화면 열기
- [ ] 관리자 이메일/비밀번호 미리 입력 가능 상태 확인
- [ ] 녹화 영역에 주소창, 북마크바, 메신저 알림이 보이지 않는다
- [ ] 마우스 커서 이동 속도를 너무 빠르지 않게 조정

## During Recording

- [ ] 검색어는 `linen`
- [ ] 상품 상세에서 `장바구니 담기`
- [ ] 장바구니에서 `주문서로 이동`
- [ ] 체크아웃에서 `주문하기`
- [ ] 주문번호가 보이면 바로 관리자 탭으로 전환
- [ ] 관리자 로그인
- [ ] `주문` 이동 후 방금 주문 강조
- [ ] `상품` 이동 후 `New product` 클릭

## If Something Fails

- [ ] 검색 결과가 이상하면 `/products/linen-bed-set` 로 바로 이동
- [ ] 체크아웃 입력이 꼬이면 브라우저 새로고침 후 다시 진행
- [ ] 주문이 안 보이면 관리자 `주문` 화면 새로고침
- [ ] 관리자 로그인 실패 시 `/admin` 대시보드까지만 보여주고 네비게이션 존재를 강조

## After Recording

- [ ] 주문번호가 화면에 한번은 분명히 보였다
- [ ] 관리자 주문 목록에서 같은 주문을 확인했다
- [ ] `상품 > New product` 폼 진입 장면이 들어갔다
- [ ] 전체 러닝타임이 `65초 이하` 다
