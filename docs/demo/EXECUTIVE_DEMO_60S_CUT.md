# EXECUTIVE DEMO 60S CUT

## Timeboxed Sequence

### 0:00-0:06

- 홈 진입
- 검색창 클릭
- `linen` 입력
- `검색`

자막:
`고객은 상품을 검색하고`

### 0:06-0:14

- `linen-bed-set` 클릭
- 상품 상세 진입

자막:
`상세 페이지에서 장바구니에 담습니다`

### 0:14-0:24

- `장바구니 담기`
- 우상단 장바구니 진입
- `주문서로 이동`

자막:
`장바구니에서 주문으로 이어지고`

### 0:24-0:35

- 체크아웃 필수 입력
- `주문하기`
- 주문 완료 화면 확인

자막:
`주문이 완료되면`

### 0:35-0:48

- 관리자 로그인 탭 전환
- `로그인`
- `주문` 클릭
- 방금 주문 확인

자막:
`관리자는 관리자 화면에서 즉시 확인합니다`

### 0:48-1:00

- `상품` 클릭
- `New product` 클릭
- 상품 폼 진입 상태에서 종료

자막:
`같은 콘솔에서 상품 운영까지 이어집니다`

## Hard Rules

1. 회원가입 화면은 보여주지 않는다
2. 고객 로그인 화면은 보여주지 않는다
3. 관리자 주문 화면까지만 보여줘도 메시지는 충분하다
4. 상품은 저장하지 않고 `New product` 진입까지만 보여준다

## Visual Priority

1. 주문 완료 화면의 주문번호
2. 관리자 주문 화면의 같은 주문번호
3. `New product` 버튼과 폼 진입
